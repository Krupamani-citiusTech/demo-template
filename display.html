CSS Dispaly Property

block, Inline, Inline-block, none

Displays an element as an inline element (like ). Any height and width properties will have no effect

Displays an element as a block element (like 'p tag'). It starts on a new line, and takes up the whole width

Displays an element as an inline-level block container. The element itself is formatted as an inline element, but you can apply height and width values

The element is completely removed
